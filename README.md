# Backyard Flyer Project

## How to run ? 

1. Setup environment
	
	`conda activate fcnd`

2.	Run the simulator.

3. Run the code

	`python backyard_flyer.py`
	
	You should see output like this in the terminal and **the drone should fly in a square on the simulator.**
	
	```
	Logs/TLog.txt
	Logs/NavLog.txt
	starting connection
	arming transition
	takeoff transition
	Setting Home
	waypoint transition
	target position [10.0, 0.0, 3.0]
	waypoint transition
	target position [10.0, 10.0, 3.0]
	waypoint transition
	target position [0.0, 10.0, 3.0]
	waypoint transition
	target position [0.0, 0.0, 3.0]
	landing transition

	```
